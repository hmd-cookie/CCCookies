# CCCookies

Homemade chocolate chip cookies, baked with love.

A 3D interactive e-commerce website built with Next.js 15, React Three Fiber, Tailwind CSS, and Zustand.

## Stack

- **Framework:** Next.js 16 (App Router)
- **3D Rendering:** React Three Fiber + Three.js
- **Styling:** Tailwind CSS v4
- **State:** Zustand (persisted cart)
- **Payments:** Stripe (Phase 3)
- **Auth:** Supabase (Phase 3)
- **Email:** Resend (Phase 3)
- **Hosting:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
